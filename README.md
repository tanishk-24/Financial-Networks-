# Financial Networks

This repository contains the implementation and analysis of **Functional networks from multivariate financial time series** (based on the work/course by Anindya S. Chakrabarti).

## Overview

The project explores various properties and network representations of financial markets, drawing parallels with complex systems like the human brain. It involves analyzing stock price data, asset returns, and constructing different types of financial networks.

## Repository Structure

- **Task 0**: Empirical properties of asset returns, stylized facts, and statistical issues.
- **Task 1**: Topological isomorphisms of human brain and financial market networks.
- **Task 2**: Collective behavior of stock price movements in an emerging market (Eigendecomposition).
- **Task 3**: Fractality and Multifractal properties in stock price data.
- **Project_Description**: Contains the main project content and syllabus.
- **Research Papers**: Literature and reference papers used for the study.

## Key Topics Covered

- Properties of asset return series (Inter-day and intra-day, distribution of return, scaling laws, stationarity)
- Fractality in stock price data (Multi-scaling, Multifractal properties)
- Return correlation networks (Spectral properties, Sparsistent filtering)
- Volatility networks (GARCH model, Diebold-Yilmaz's intra-day technique)
- Granger causal networks and Co-integration networks
- Network measures on correlation networks (Centrality measures and risk)
- Filtering on financial networks (Minimum spanning tree, Spectral filtering)
- Portfolio optimization using network approaches
- Complexity of interactions and Multi-layered networks (Stocks and bonds)
